## What is Sistema de Mantenimientos?

Sistema de Mantenimientos is a PHP-based web application designed to help organizations track and manage equipment maintenance across multiple facilities. It provides a centralized interface for registering devices, scheduling preventive and corrective maintenance, assigning technicians, and monitoring equipment health over time.

## Key features

<CardGroup cols={2}>
  <Card title="Equipment registry" icon="computer">
    Register computers and peripherals with brand, type, entry date, and room assignment. Track whether each device is operational or under maintenance.
  </Card>
  <Card title="Maintenance scheduling" icon="calendar">
    Schedule preventive and corrective maintenance tasks. The system automatically calculates the next maintenance date based on a 180-day interval.
  </Card>
  <Card title="Multi-location support" icon="building">
    Organize equipment across multiple sedes (locations) and salas (rooms). Each room belongs to a specific sede, enabling hierarchical tracking.
  </Card>
  <Card title="Technician management" icon="user-gear">
    Register monitores (technicians) and assign them to specific maintenance tasks. Track who performed each maintenance job.
  </Card>
  <Card title="Maintenance history" icon="clock-rotate-left">
    View the full maintenance history for any individual piece of equipment, including start dates, end dates, and work descriptions.
  </Card>
  <Card title="Status tracking" icon="circle-check">
    Equipment status automatically updates when maintenance begins (set to "in maintenance") and when maintenance is completed (set to "operational").
  </Card>
</CardGroup>

## System modules

The application is organized into six core management modules:

| Module | URL | Description |
|--------|-----|-------------|
| Brands (Marcas) | `/index.php` | Manage equipment brands |
| Locations (Sedes) | `/pages/sedes.php` | Manage facility locations |
| Rooms (Salas) | `/pages/salas.php` | Manage rooms within locations |
| Technicians (Monitores) | `/pages/monitores.php` | Manage maintenance technicians |
| Equipment (Equipos) | `/pages/equipos.php` | Manage devices and computers |
| Maintenance | `/pages/mantenimientos.php` | Create and track maintenance records |

## Technology stack

- **Backend**: PHP 8.2+ with MySQLi extension
- **Database**: MySQL / MariaDB 10.x
- **Frontend**: Tailwind CSS (CDN) + Bootstrap 4
- **JavaScript**: jQuery 3.3 + Popper.js

<Note>
  This system uses direct SQL queries via MySQLi. Ensure your MySQL user has appropriate privileges (`SELECT`, `INSERT`, `UPDATE`, `DELETE`) on the `mantenimientos` database.
</Note>
